## 프로젝트 주제

1. ### 반려식물에 관한 여러 서비스를 제공하는 커뮤니티 웹사이트
    - 나에게 맞는 반려식물 추천 테스트
    - 스토리 게시판(일반회원인 식집사끼리 식물 관련 일상/일지 등 소통, 팔로우/팔로잉 가능)
    - 식고수 질문/답변 게시판(식고수에게 식집사들이 질문하고 답변을 통해 전문지식을 얻을 수 있음, 팔로우/팔로잉 가능)
    - 식고수 신청(신청을 관리자가 승인하면 식고수로 활동할 수 있음)
